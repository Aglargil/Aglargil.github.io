### 3.2 OBVP (Optimal Boundary Value Problem)

#### 3.2.1 建模

目标：$\min J_{\sum} = \sum_{k=1}^{3} J_k, J_k = \frac{1}{T}\int_0^T j_k(t)^2dt$
状态: $s_k = (p_k, v_k, a_k)$
输入: $u_k = j_k(t)$
系统模型: $\dot{s}_k = f_s(s, u) = (v, a,j)$
